기술 스택

HTML, CSS, JavaScript, React, Tailwind, Git등 다양한 기술을 사용한 경험이 있습니다. 저는 최신 프런트 엔드 프레임워크와 라이브러리에 능숙하며 Git을 사용하여 프로젝트에서 공동 작업을 수행할 수 있는 능력에 자신이 있습니다. 특히 최신 ES6 구문 을 사용하는데 능숙하여 깨끗하고 효율적이며 읽기 쉬운 코드를 작성할 수 있습니다.

  • Html Css Js

    HTML, CSS 및 JavaScript에 대해 확실히 이해하고 있으며 시각적으로 매력적이고 기능적이며 반응이 빠르고 사용자 친화적인 웹 사이트를 만들 수 있습니다

  • React

    React에 능숙하지는 않지만 다양한 기능을 활용할줄알고 동적인 UI를 구현하고 또한 React Router, Redux와 같은 라이브러리와 함께 작업할 수 있으며, 최신 기술 동향에 대한 지속적인 관심으로 업무에 큰 도움을 줄 수 있습니다.

  • Tailwindcss

    Tailwind CSS에 대한 초급 수준의 지식이 있지만 프레임워크를 효과적으로 사용하여 사용자 경험을 우선시하는 잘 디자인되고 반응이 빠른 웹 사이트를 만드는 능력에 자신이 있습니다.

  • Git

    신입 개발자로서 경험이 많지 않을 수 있지만 버전 관리를 위해 Git을 사용하고 팀과 함께 프로젝트를 공동 작업하는 데 능숙합니다. 저는 지속적으로 제 기술을 향상시키고 최신 개발 관행을 따라잡기 위해 최선을 다하고 있습니다.
